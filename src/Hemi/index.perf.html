What Is "Hemi"?

The Hemi JavaScript Framework (v. 3.0) is a services-oriented Web Page framework. It is the third major version of Engine for Web Applications project.

About the Hemi JavaScript Framework

Where is version 2? Version 2 is Engine for Web Applications.

What is the difference between Hemi and Engine? New features, a better class loader, a more generic namespace - many things. All of the configuration fat was cut.

Will it work in my browser? Run the feature tests

Are there any developer tools? The Framework Profiler is a good place to start. The DWAC Designer is another rich development toolset. You can access DWAC by registering with this site, or using the Core Web project.

Who Uses Hemi?

Facets of Hemi are used in a number of products, including JavaScript Profiler, libXmlRequest, and some enterprise monitoring solutions.
