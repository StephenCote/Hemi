Hemi JavaScript Framework

Hemi.util.url

  • Hemi JavaScript Framework
  • API Index

Description

The URL class parses URIs into individual parts. The class also supports resolving relative paths against specified URLs.

Class Index

  • static Hemi.util.url: Utility for parsing URLs.
  • Hemi.util.UrlInstance: On object representation of a specified URL.
