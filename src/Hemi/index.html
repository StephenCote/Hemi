Hemi Framework - v3.5

What Is "Hemi"?

The Hemi JavaScript Framework (v. 3.5) is a services-oriented Web Page framework. It is the third major version of Engine for Web Applications project.

Setup

  • Download the latest version: Download v3.5 from GitHub
  • Detailed instructions are incuded in the Hemi Wiki.

About the Hemi JavaScript Framework

New features in v 3.5 include the Hemi JSON RPC and Hemi JSON RPC Cache services.

Mini FAQ

  • Will it work in my browser or device? Run the feature tests, which perform automatic build-verification testing, to verify the framework modules are operating correctly.
  • Are there any developer tools?
    • For in-page framework testing, use the The Framework Profiler.
    • Experimental: Try an inline non-persistent Framework Designer: For prototyping new Hemi projects, tasks, templates, components, and fragments. Login required to save data on server.
  • Are there any monitors for the Monitor Service? Yes. Performance, Behavior (aka: IMNMotion), and Experience (aka: Raccoon) monitors are available, but require a correctly configured server for configuration, shredding, aggregation, and reporting. These modules are not presently made available.
